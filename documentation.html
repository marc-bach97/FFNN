#Interaktionen

  • Training und Tests: Auf dieser Seite können Sie Daten mit oder ohne Rauschen generieren, indem Sie die Stichprobengröße angeben. Anschließend können Sie das Modell erstellen und verborgene Schichten definieren. Wenn Sie auf "Modell generieren und trainieren" klicken, wird das Modell anhand der von Ihnen generierten Daten erstellt und trainiert , dann wird eine Prüfung durchgeführt und angezeigt. Sie können das Modell auch speichern (dadurch werden zwei Dateien heruntergeladen, die erste enthält die Modellstruktur und die zweite die Gewichte).
  • Datenbeispiele: Auf dieser Seite können Sie sehen, wie Daten generiert werden. Sie können die Anzahl der Proben auswählen, die Sie generieren möchten. Es werden zwei Datensätze generiert, einer ohne Rauschen und der zweite mit Rauschen.
  • Resultate: Auf dieser Registerkarte gibt es zwei Abschnitte, einen für Daten ohne Rauschen und den anderen für die Ergebnisse von Daten mit Rauschen.Für jeden Abschnitt können Sie die Ergebnisse sehen und das vorab trainierte Modell mit allen Details laden. Außerdem wird ein automatischer Test durchgeführt und angezeigt

#Entwicklung

TensorFlow.js

TensorFlow.js ist eine Open-Source-Bibliothek, die es Entwicklern ermöglicht, Maschinenlernmodelle direkt im Browser oder auf Node.js zu definieren, zu trainieren und auszuführen. Es handelt sich um eine Erweiterung des TensorFlow-Frameworks, das entwickelt wurde, um die Möglichkeiten des Maschinenlernens für JavaScript-Entwickler zugänglich zu machen. TensorFlow.js ermöglicht die Erstellung von Maschinenlernmodellen für verschiedene Aufgaben, einschließlich Bild- und Spracherkennung, natürlicher Sprachverarbeitung und mehr.


TensorFlow.js Visualization

tfjs-vis (TensorFlow.js Visualization) ist eine Bibliothek, die Visualisierungstools und -dienstprogramme für TensorFlow.js bereitstellt. Sie wurde entwickelt, um es Entwicklern zu erleichtern, das Verhalten von mit TensorFlow.js erstellten maschinellen Lernmodellen zu visualisieren und zu verstehen.


Bootstrap

Bootstrap ist ein weit verbreitetes Open-Source-Front-End-Framework zur Entwicklung von ansprechenden und mobilen Webanwendungen. Von Twitter entwickelt, bietet Bootstrap eine Sammlung vordefinierter HTML-, CSS- und JavaScript-Komponenten, die Entwicklern dabei helfen, konsistente und visuell ansprechende Benutzeroberflächen zu erstellen.
